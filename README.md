# Broken

Broken is a short project (for educational purposes) aiming to interact with CS2D game memory.

## Features
- Read some values of the current player
- Flashbang hack

### Incoming features
- Anti-fog / Night-Vision
- Scope-hack (invisible scoping)
- Radar
- Free-camera
- Speed-hack
- Aimbot
- Simple UI
